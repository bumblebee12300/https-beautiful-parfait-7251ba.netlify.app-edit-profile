# Task Manager App

A React + Vite task manager web application with add, delete, and complete task functionality.

## Features
- Add new tasks with validation
- Delete tasks
- Toggle tasks as completed
- Responsive layout and accessible controls

## Getting Started

1. Open the `task-manager-app` folder in VS Code.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the app in your browser using the URL shown by Vite.

## Copilot Customization

- `.github/copilot-instructions.md` contains custom project guidelines.
- `.github/agents/Reviewer.agent.md` defines a custom code review agent.

## Notes

This project is intentionally simple and uses React state only. It is a good base for adding features such as persistence, filters, or dark mode.
